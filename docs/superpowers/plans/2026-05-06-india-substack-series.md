# India Through Its States — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Publish 28 long-form Substack posts covering every Indian state, ordered geographically from Northeast to West, each built around a single anchor reframe.

**Architecture:** Each state runs a 5-day pipeline — Reframe Discovery gates everything; sections are generated modularly; the Reframe Test gates publication. A Reframe Log accumulates after each post to prevent repetition and generate cross-post callbacks.

**Publication cadence:** 1 post per week · 28 weeks total · ~7 months
